Program directory box for the audience pages. Fully visible (no hover-to-reveal): photo, plain label, full official name, one factual line, and tags.

```jsx
<ProgramCard
  label="Hinman CEOs"
  fullName="Hinman CEOs Program"
  line="Living-learning community for student entrepreneurs"
  tags={["Undergraduate", "Community & space"]}
/>
```

Pass `image` for a real photo; omit it for a hatched "Photo" placeholder. No stage glyphs on these boxes — they stay clean for filtering. Drives the Students-page tag filter.
